## Installation

#### Firefox

Patio is available on the [official Mozilla Add-ons store](https://addons.mozilla.org/en-US/firefox/addon/patio/)



## Build from source

#### Requirements

- npm


#### Build for Chrome

```sh
npm install                # Install dependencies
npm run release:chrome     # Build
```


#### Build for Firefox

```sh
npm install                # Install dependencies
npm run release:firefox    # Build
```
